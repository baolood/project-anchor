# PROJECT_ANCHOR_EARLY_DEMO_RECRUITING_LIST_V1

## 1. 任务结论

本文件用于定义 Project Anchor Early Demo 第一轮真实用户招募清单。

当前目标不是扩大宣传，也不是公开上线，而是找到 3-5 个真实 crypto 交易者，看 20 分钟只读演示，验证他们是否存在“交易纪律执行困难”这个真实痛点。

本轮只做招募清单与访谈记录规则，不改产品功能、不接真实资金、不接真实 API key、不开放真实交易。

## 3. 第一轮招募目标

第一轮只招募 3-5 人。

人数不追求多，追求真实、具体、有交易经历。

合格对象必须满足至少 2 条：

- 有 crypto 交易经验
- 交易过 BTC / ETH / 主流币
- 曾经因为冲动交易、追涨杀跌、加仓、频繁操作亏过钱
- 承认自己有纪律执行问题
- 有自己的交易规则，但经常无法执行
- 不喜欢黑箱交易机器人
- 愿意看一个不接资金、不下单、不承诺收益的只读 Demo

---

## 4. 优先找的人

优先级从高到低：

### P0：最优先

- 你现实中认识的 crypto 交易者
- 曾经和你聊过交易亏损、冲动操作、风控失败的人
- 有 BTC / ETH 现货交易经验的人
- 做过定投、网格、低频交易但经常手动破坏规则的人

### P1：可以找

- Telegram / Discord 小群里的熟人
- X / Twitter 上互动过的交易者
- LinkedIn 上对交易纪律、风险控制、AI 工具感兴趣的人
- 朋友介绍的 crypto 交易者

### P2：暂不优先

- 完全陌生的大 V
- 高频量化团队
- 机构交易团队
- 只想找暴利信号的人
- 只问“每天能赚多少”的人

---

## 5. 暂时不要找的人

第一轮不要找：

- 重杠杆玩家
- 高频交易者
- 合约赌徒
- 只关心暴利信号的人
- 想让系统直接代操资金的人
- 期待“AI 自动赚钱”的人
- 不愿意承认自己有纪律问题的人
- 没有任何 crypto 交易经验的人

原因：

这些人会把 Project Anchor 错看成交易机器人或收益工具，容易干扰第一轮验证。

## 10. 目标用户判定问题

联系前或演示前，先问 3 个筛选问题：

1. 你过去 6 个月是否交易过 crypto？
2. 你是否有过冲动交易、追涨杀跌、加仓或破坏计划的经历？
3. 如果一个系统在下单前强制你检查规则，你会觉得有帮助，还是觉得麻烦？

判断：

- 3 个问题都能具体回答：优先演示
- 只关心收益率：暂缓
- 不承认纪律问题：暂缓
- 只想自动赚钱：拒绝进入第一轮

---

## 11. 20 分钟演示后必须问的问题

每次演示后固定问 5 个问题：

1. 你最近一次冲动交易是什么情况？
2. 如果系统在下单前强制你检查规则，你会觉得有帮助还是麻烦？
3. 你最希望它拦住哪类操作？
4. 你愿意为这种“纪律执行工具”付费吗？如果愿意，每月大概多少？
5. 你敢不敢把真实交易 API 接进这种系统？如果不敢，卡点是什么？

## 12. 访谈记录字段

每次访谈后只记录以下字段：

- 日期
- 访谈对象代号
- 来源渠道
- 是否有 crypto 交易经验
- 是否承认有纪律问题
- 最近一次冲动交易描述
- 对 Demo 的第一反应
- 最想被系统拦住的操作
- 是否愿意继续试用
- 是否愿意讨论付费
- 可接受价格区间
- 是否愿意介绍别人
- 最大顾虑
- 结论：strong / medium / weak / not target

禁止记录：

- 真实姓名
- 交易所账号
- API key
- 私人资产金额
- 身份证件
- 银行信息
- 钱包私钥 / 助记词

---

## 13. 强信号判断

强信号包括：

- 用户主动讲具体亏损经历
- 用户说“我确实需要这个”
- 用户问什么时候可以试用
- 用户愿意留下联系方式
- 用户愿意讨论价格
- 用户愿意介绍同类朋友
- 用户明确说不想要交易机器人，但想要纪律约束

中等信号包括：

- 觉得方向有意思
- 愿意再看下一版
- 认可风险控制价值
- 但还不确定是否付费

弱信号包括：

- 只说“挺有意思”
- 只问收益率
- 只问能不能自动赚钱
- 不愿意讲自己的交易问题
- 不愿意继续试用

---

## 14. 第一轮成功标准

第一轮不要求很多人付费。

3-5 人演示后，只要满足以下条件，就算继续推进有依据：

- 至少 2 人承认自己有纪律执行问题
- 至少 1 人愿意继续试用
- 至少 1 人愿意讨论付费或价格
- 至少 1 人愿意介绍类似用户
- 没有人把 Demo 误解为真实资金托管系统

第一轮失败标准：

- 大多数人只关心收益率
- 没有人承认纪律问题
- 没有人愿意继续试用
- 用户普遍无法理解产品价值
- 用户全部期待自动交易或暴利信号
